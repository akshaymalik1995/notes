# Setting up development environment for ES6

1. `npm init -y`. This will create a package.json file within your folder. This will define it as a nodejs project.

2. So next thing you want to do is install webpack and webpack-cli. I would recommend you to install webpack as a dependency and webpack-cli as a development dependency.
`npm i webpack --save`

`npm i webpack-cli --save-dev`

3. In the next step create a folder and name as you like. Here we have created an `src` folder and created a  `index.js` file where you’ll write you javascript code.
And inside `index.js` add a simple javascript code to check if its working or not. 

`console.log("Life is beautiful")`


4. Now in the root folder create a file called `webpack.config.js` which will be our webpack configuration file.

```js
module.exports = {
  
  entry: path.resolve(__dirname, 'app'),
  output: {
    path: path.resolve(__dirname, 'build'),
    filename: "bundle.js"
  },
```

5. Now everything is done. We have to run and check if it is working or not. For this step, you can now go to your `package.json` file. And search `"scripts"` and add `“build”: “webpack --watch”` and your script should look like this now.

```js
  "scripts": {
    "start": "webpack --watch --mode development",
    "build": "webpack --mode production"
  },
```

6. Create a folder `public` in the root folder. In that folder, create a file `index.html` that has our `bundle.js` loaded. Here is our HTML file file should looks like.

```html

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-  scale=1.0">
  <title>ES6 Development Env</title>
</head>
<body>
  <!-- Content Goes here -->
</body>
<script src="../build/bundle.js" type="text/javascript"></script>
</html>
```

`build` is folder webpack creates so remember to keep bundle out for `bundle.js`

7. Now run `npm run build`

In step 5 we used `“build”: “webpack --watch”` which will watch for our changes in the js file and update it accordingly.

**You have successfully added webpack and now let's add BabelJS.**

8. Add `mode: “development”`, in your webpack config file. 



To add babel run:

`npm install --save-dev @babel/core babel-loader babel-preset-env`

Babel-core and babel-loader help to transpile ES6 code to ES5 so that our browser can understand our written ES6 and babel-preset-env lets us use advanced features of ECMA script in our web apps.

Creare a `.babelrc` file and add the following content:

```
{
    "presets": ["env"]
}
```

9. After you add those packages add module to module.exports object like this:

```js
const path = require('path');
module.exports = {
  
  entry: path.resolve(__dirname, 'app'),
  output: {
    path: path.resolve(__dirname, 'build'),
    filename: "bundle.js"
  },
  module: {
    rules: [
    {
      test: /\.js$/, //transpile all .js files
      exclude: /(node_modules)/, //exclude node modules
      use: ['babel-loader'],
    }
   ]
    }
   
}



```

10. Install the webpack dev server.

`npm i webpack-dev-server --save-dev`

Modify the `package.json` file.

```js
"scripts": {
    "start": "webpack-dev-server --mode development",
    "build": "webpack --mode production"
  },
  
  
```

Add devServer to the `webpack.config.js` file.

```js
const path = require('path');
module.exports = {
  
  entry: path.resolve(__dirname, 'app'),
  output: {
    path: path.resolve(__dirname, 'build'),
    filename: "bundle.js"
  },
  module: {
    rules: [
    {
      test: /\.js$/, //transpile all .js files
      exclude: /(node_modules)/, //exclude node modules
      use: ['babel-loader'],
    }
   ]
    },
    devServer : {
        port : 3000,
        contentBase : path.resolve(__dirname, 'build')
        
       }
}


```
Open `locallhost:3000` in the browswer.


10. Now go to terminal and fire up `npm run build`.
