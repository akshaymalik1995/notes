
**What Is The React Context API?**

1. React Context API is a way to essentially create global variables that can be passed around in a React app.
2. The Context API provides a way to pass data through the component tree without having to pass props down manually at every level.
3. What the context API does is that it makes a data layer which covers the entire application.
4. There is only 1 main store house where all the data is passed to and from where all the data is extracted from.
