## Const and Let

`const` and `let` are used for block-level scoping.

```js
const name;
// Incorrect syntax since we cannot initialize a variable using const

const name = "Akshay"
name = "Rahul"
// Incorrect syntax since we cannot change a variable created with const keyword

let name;
// Correct Syntax
```

## Arrow functions

```js
// Normal function
function sayHello(message){
console.log(message)
}

// Arrow function

const sayHello = (message) => {
console.log()
}

const sayHello = message => console.log();
```
