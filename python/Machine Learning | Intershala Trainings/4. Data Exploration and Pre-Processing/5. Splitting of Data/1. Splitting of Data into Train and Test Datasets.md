## Train and Test Datasets


The data on which a model is built is called **Training Data**. This data is used by the model to learn.

Once the model is trained, it is examined as to how well it has learnt using another subset of the original data which is called **Test Data**. The model predicts the target variable values for the test data and then the predicted values are compared with actual values and checked as to how many of them were correctly predicted.

Ideal Split Ratio is 70:30. However, it depends on the size of the dataset and the problem statement.

The `sklearn.model_selection` library is used to split data.

```
from sklearn.model_selection import train_test_split
x_train, x_test, y_train, y_test = train_test_split(X, Y, test_size=0.3)
```
