# Binning and Creation of Dummy Variables

## Creation of Dummy Variables

It refers to the process of transforming a categorical variable into a set of numerical or boolean variables, called dummy variables, each of which has values 0 and 1.

The number of dummy variables created for a categorical variable depends on the number of unique values or levels that categorical variable has.

If a categorical variable has `n` levels, then `n-1` dummy variables are required.

**Why do we need dummy variables?**

1. Regression modelling requires all the independent variables to be numerical variables.
2. By creating a dummy variable, we transform a categorical variable into a numerical variable.
3. After creating required dummy variables for a categorical variable, categorical variable is dropped from the dataset.
4. The `get_dummies` function of the pandas library creates the dummy variables for different levels of the categorical variable and then drops it.


![image](https://user-images.githubusercontent.com/55041489/130383202-ec67f7d9-0618-4fce-9036-e4d8d9e501be.png)

```
df1 = pd.get_dummies(df, columns=['Condition of the House'], drop_first=True)
```




```
df1 = pd.get_dummies(df1, columns=['Waterfront View'], drop_first=True)
df1 = pd.get_dummies(df1, columns=['Ever Renovated'], drop_first=True)
```

## Binning

If the number of levels for a categorical variable is more (say > 20), then bin these levels into fewer groups before creating the dummy variables.

For example, for a variable 'zipcode', we may need as many as 70 dummy variables if the unique values are 70. We can zipcodes with similar average Sale Price into groups.

Binning can be used both for numerical and categorical variables.

Binning is also an example of variable transformation.

**Group by the table on the basis of zipcodes and get their mean value and arrange them in ascending order**

```
zip_table = df1.groupby("Zipcode").agg({'Sale Price' : 'mean'}).sort_values('Sale Price', ascending=True)
```

![image](https://user-images.githubusercontent.com/55041489/130387041-bd3a473d-d1bd-4413-a1f7-a37a0a8265cc.png)

**Bin the zipcode table into various groups**

```
zip_table['Zipcode_Group'] = pd.cut(zip_table['Sale Price'], bins=10, labels=['Zipcode_Group_0','Zipcode_Group_1','Zipcode_Group_2','Zipcode_Group_3','Zipcode_Group_4','Zipcode_Group_5','Zipcode_Group_6','Zipcode_Group_7','Zipcode_Group_8','Zipcode_Group_9'], include_lowest=True)
```

![image](https://user-images.githubusercontent.com/55041489/130387143-798ecf30-7d32-4c9e-abfe-937ca39150fc.png)

**Drop the Sale Price column from the zipcode table**

```
zip_table = zip_table.drop(columns = 'Sale Price')
```

![image](https://user-images.githubusercontent.com/55041489/130387260-c36f0d51-356e-43ab-9119-ab58a84eaab6.png)


**Merge the zipcode table with the original table**

```
df1 = pd.merge(df1, zip_table, left_on = 'Zipcode', how='left', right_index=True)
df1 = df1.drop(columns='Zipcode')
```

![image](https://user-images.githubusercontent.com/55041489/130387532-a4595219-d736-4b57-89ce-b72668c2df12.png)

**Create dummy variables for Zipcode_Group**

```
df1 = pd.get_dummies(df1, columns=['Zipcode_Group'], drop_first=True)
```
