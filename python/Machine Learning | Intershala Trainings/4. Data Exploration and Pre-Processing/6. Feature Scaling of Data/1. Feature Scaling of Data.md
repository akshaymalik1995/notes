# Feature Scaling of Data

Feature Scaling is all about scaling the feature variables (i.e. all the independent variables) into the same range.



![image](https://user-images.githubusercontent.com/55041489/130432334-352b5a2c-b425-4ac1-824d-3eb1b403be7b.png)

**The variation in magnitude and range of features causes two problems:**

1. When the model is built, variables with higher magnitude and range will have more impact compared to the ones which are smaller.
2. The model might not predict properly since it is not giving equal weight to all the variables.
3. The gradient descent algorithm that is used to find the coefficients of linear regression may take a long time to converge.

## Feature Scaling ##

1. The variables are scaled to have similar magnitude and ranges so that model is not biased towards a particular variable.
2. Feature Scaling is must for those algorithms where some meaure of distance between two data points is involved like Logistic Regression, Linear Regression, K Nearest Neighbours, Principal Component Analysis, etc.
3. However, Feature scaling is not required for tree-based algorithms like Random Forest, Decision Trees, etc.


## Different Techniques of Feature Scaling

**Standardisation**
It rescales the feature values so that they have the properties of a Standard Normal Distribution with mean as 0 and the standard deviation of 1.

![image](https://user-images.githubusercontent.com/55041489/130432817-a0ccdf1d-5849-4646-b45e-8367a7c37e4e.png)

**Min Max Scaling**
One of the simplest for scaling. The value range for the transformed variables lies between [0, 1]


![image](https://user-images.githubusercontent.com/55041489/130433177-e9c10e12-fa15-4b9a-82c8-1a46fd595a9b.png)

**Normalization**

The range is fixed from -1 to 1.

![image](https://user-images.githubusercontent.com/55041489/130433475-faf8feb4-73ab-43f5-8e28-2c76f635a9e9.png)

## Scaling using python

1. Standardisation method should be used for scaling the feature variables when a linear regression is built. 
2. A linear regression model assumes the input variables to be normally distributed.
3. The preprocessing library of 'sklearn' can be used to do the standardisation.

```
from sklearn import preprocessing
scale = preprocessing.StandardScaler()
x_train = scale.fit_transform(x_train)
x_test = scale.fit_transform(x_test)
```
