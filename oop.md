### Deleting attributes and objects

Any attribute of an object can be deleted anytime, using the `del` statement.

```
del akshay.age                   #deleting attribute
del akshay.get_datq              #deleting method
del akshay                       #deleting the object
```

When we do `akshay = Person()`, a new instance object is created in memory and the name `akshay` binds with it.
